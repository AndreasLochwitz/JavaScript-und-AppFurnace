Kommentare

Kommentare in JavaScript können sowohl einzeilig wie auch mehrzeilig (wie es auch in Java ist) angelegt werden.

Debugausgaben

Mittels der Funktion console.log kann man den Inhalt von Variablen in der JavaScript-Konsole des Browsers ausgeben.

Variablen und deren Sichtbarkeit

In JavaScript werden Variablen mit dem Schlüsselwort var definiert. Erfolgt die Definition außerhalb einer Methode, ist die Variable global sichtbar, dass heißt sie kann von jeder Funktion geändert werden. Wird die Definition innerhalb einer Funktion durchgeführt, ist die Variable nur innerhalb dieser Funktion bzw. darin erstellten anonymen Funktionen sichtbar.

Datenstruktur 1: Array

Mit Arrays können eine Menge von gleichartigen Variablen verwaltet werden, wo es nicht notwendig ist, einen direkten Zugriff auf die einzelnen Elemente zu haben. Im Gegensatz zu Java können Arrays in JavaScript zur Laufzeit beliebig verändert werden.

Datenstruktur 2: Hash-Array

Mit Hash-Arrays können Daten als Key-Value Paare verwaltet werden. Der Schlüssel (Key) muss einmalig innerhalb des Hash-Arrays sein und dient der Referenzierung (Zugriff) auf die Daten (Value). Hash-Arrays werden auch als assoziative Arrays bezeichnet.

Zeitverzögerte Ausführung und Timer

Mit Hilfe der Funktion setTimeout kann die Ausführung von Programm-Code auf einen späteren Zeitpunkt verschoben werden.

Mit der Funktion setInterval kann es realisiert werden, dass Programm-Code immer wieder aufgerufen wird. Mit der Funktion clearInterval kann ein zuvor mit setInterval gestartetes Interval wieder beendet werden. Dazu ist es notwendig, den Rückgabewert von setInterval hier anzugeben.

Zufallszahlen

Mit Hilfe der Funktion Math.random können Zufallszahlen generiert werden. Der Rückgabewert dieser Funktion ist eine Fließkommazahl mit einem Wert zwischen 0 und 1.

Externe Dateien laden

Mit Hilfe der LazyLoad.js Bibliothek können zur Laufzeit dynamisch weitere JavaScript und CSS Dateien geladen werden, die das Aussehen oder die Funktionalität erweitern und verändern.
